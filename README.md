# ⚽ Escala Jogador - Time de Futebol

Exercício do curso de JavaScript onde o objetivo era criar uma página web 
para gerenciar a escalação de um time de futebol.

## 📋 O que foi pedido no exercício

- Escalar um jogador informando nome, posição e número da camisa
- Remover um jogador informando o número da camisa

## 🚀 O que foi implementado além do exercício

- Menu de confirmação antes de escalar um jogador
- Menu de confirmação antes de remover um jogador
- Validação de campos vazios
- Validação de número de camisa inválido
- Feedback visual ao usuário em cada ação

## 🛠️ Tecnologias utilizadas

- HTML
- JavaScript (DOM puro)
